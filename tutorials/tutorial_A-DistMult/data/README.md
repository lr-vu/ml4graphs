# WN18RR

WordNet dataset

`wn18rr-train.nt.gz`
- training split; RDF version of original 

`wn18rr-test.nt.gz`
- test split; RDF version of original 

`wn18rr-valid.nt.gz`
- validation split; RDF version of original
